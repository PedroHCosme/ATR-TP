\doxysubsection{Detailed Description}
Classe auxiliar para gerenciar temporizadores assíncronos. 

Permite agendar a execução de callbacks após um determinado período de tempo, sem bloquear a thread de execução, utilizando o io\+\_\+context do Boost.\+Asio.
