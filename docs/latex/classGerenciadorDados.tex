\doxysubsection{Detailed Description}
Classe que implementa o padrão Monitor para gerenciar o acesso aos dados compartilhados. 

Esta classe encapsula o buffer circular de dados dos sensores, bem como o estado atual do veículo e os comandos do operador. Fornece métodos thread-\/safe para leitura e escrita, garantindo a integridade dos dados em um ambiente multithread.
