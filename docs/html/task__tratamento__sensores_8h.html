Calcula o próximo valor da Média Móvel Exponencial (EMA).

Filtro digital simples utilizado para suavizar leituras ruidosas dos sensores. A fórmula utilizada é: EMA_atual = (Valor_atual - EMA_anterior) * K + EMA_anterior.

media_anteriorO valor da EMA calculado no passo anterior.
Returns

Tarefa produtora responsável pela leitura e pré-processamento dos sensores.

Esta função simula a aquisição de dados consultando o estado "real" na simulação física, aplica ruído gaussiano para simular imperfeições dos sensores reais, e aplica filtros (como EMA) antes de disponibilizar os dados no buffer compartilhado.

Parameters
gerenciadorDadosReferência para o objeto GerenciadorDados onde os dados serão escritos.
simulacaoReferência para a simulação física de onde a "verdade" é lida.
id_caminhaoID do caminhão cujos sensores estão sendo lidos.
