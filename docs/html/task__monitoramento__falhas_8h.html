Tarefa responsável por monitorar a saúde do sistema e tratar falhas.

Verifica periodicamente os dados dos sensores e o estado da simulação em busca de anomalias (ex: superaquecimento, falhas elétricas). Caso detecte um problema, sinaliza o evento correspondente para que o sistema possa reagir (ex: parada de emergência).

Parameters
gerenciadorDadosReferência para o gerenciador de dados.
simulacaoReferência para a simulação física (para injeção/verificação de falhas reais).
eventosReferência para o sistema de sinalização de eventos.
