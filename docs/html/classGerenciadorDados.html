Detailed Description

Classe que implementa o padrão Monitor para gerenciar o acesso aos dados compartilhados.

Esta classe encapsula o buffer circular de dados dos sensores, bem como o estado atual do veículo e os comandos do operador. Fornece métodos thread-safe para leitura e escrita, garantindo a integridade dos dados em um ambiente multithread.

◆ getDados()

DadosSensores GerenciadorDados::getDados ( )

Recupera o dado mais antigo do buffer circular.

Obtém dados de sensores do buffer circular. Esta função é segura para threads. Se o buffer estiver vazio, a thread que chama aguardará até que haja dados disponíveis. Notifica um produtor em espera após remover os dados.

Atua como Consumidor. Bloqueia a thread se o buffer estiver vazio até que novos dados estejam disponíveis.

void GerenciadorDados::setDados ( const DadosSensores & dados)

Insere novos dados de sensores no buffer circular.

Adiciona dados de sensores ao buffer circular. Esta função é segura para threads. Se o buffer estiver cheio, a thread que chama aguardará até que haja espaço. Notifica um consumidor em espera após adicionar os dados.

Atua como Produtor. Se o buffer estiver cheio, sobrescreve o dado mais antigo. Notifica consumidores aguardando dados.
